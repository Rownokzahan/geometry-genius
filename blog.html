2. What is DOM and what is the purpose of DOM?

The DOM stands for Document Object Model. DOM creates a structured representation of a web page in the form of a tree-tdke structure, where each element on the page is represented as a node in the tree.
The purpose of DOM is to be able to access the content, structure, and style for every element of web pages and change or modify if needed.

2. How will you select HTML elements using DOM? Name the DOM methods.

HTML elements can be selected by using five methods bellow

1. Tag Name :  getElementsByTagName()
2. Class Name : getElementsByClassName()
3. ID : getElementById()
4. Query Selector : querySelector()
5. Query Selector All   : querySelectorAll()

3. What is event bubble?

When an event is applied on an element (like click or mouseover), after the event has completed it continues to "bubble up" through the hierarchy of parent elements and triggers the event listeners of parent elements. This behavior is called event buble.

4. What is a callback function and why will you use it?

A callback function is a function that is given as an argument to another function.It can be called by the receiving function at a certain point in its execution.
I will use callback function for cleaner, more efficient code that is easier to modify and maintain over time. For example If some line of similer line of codes needs to happen for multiple function instead of repeating them here the callback function can be used.
